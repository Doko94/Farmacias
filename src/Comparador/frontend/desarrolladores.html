PRIMEROS PASOS

Autenticación y formato

La versión actual no requiere una clave API. Todas las respuestas utilizan JSON y codificación UTF-8. En producción se recomienda añadir autenticación, cuotas y límites por cliente.

Base URLBackend no configurado
GET/api/search

Buscar productos

Obtiene ofertas que coinciden con nombre, marca, dosis, presentación o principio activo.

Parámetros
qstring · requerido · mínimo 2 caracteres
regionstring · predeterminado: Tarapaca
communestring · predeterminado: Iquique
limitinteger · 1 a 200
curl "{BASE}/api/search?q=aciclovir%20200%20mg&region=Tarapaca&commune=Iquique"
GET/api/bioequivalents

Consultar bioequivalentes

Devuelve alternativas informadas como bioequivalentes, junto con ahorro absoluto y porcentual.

fetch("{BASE}/api/bioequivalents?q=aciclovir&region=Tarapaca&commune=Iquique")
  .then(response => response.json())
  .then(data => console.log(data.results));
GET/api/history/{pharmacy}/{sku}

Historial de precios

Entrega los precios almacenados para una oferta específica, ordenados por fecha de captura.

curl "{BASE}/api/history/Ahumada/93445?region=Tarapaca&commune=Iquique"
POST/api/treatments/monthly-cost

Calcular un tratamiento

Calcula cuántos envases se necesitan y el costo total para un periodo.

curl -X POST "{BASE}/api/treatments/monthly-cost" \
  -H "Content-Type: application/json" \
  -d '{
    "region": "Tarapaca",
    "commune": "Iquique",
    "days": 30,
    "items": [{
      "query": "paracetamol 500 mg",
      "units_per_dose": 1,
      "doses_per_day": 2,
      "units_per_package": 20
    }]
  }'
POST/api/recipes/optimize

Optimizar una receta

Distribuye productos entre farmacias cuando el ahorro compensa dividir la compra.

curl -X POST "{BASE}/api/recipes/optimize" \
  -H "Content-Type: application/json" \
  -d '{
    "region": "Tarapaca",
    "commune": "Iquique",
    "pickup": true,
    "minimum_split_savings": 2000,
    "items": [
      {"query": "paracetamol", "quantity": 2},
      {"query": "ibuprofeno", "quantity": 1}
    ]
  }'
GET/api/analytics/summary

Resumen analítico

Entrega ofertas, disponibilidad, cobertura por farmacia y descuento promedio para una ubicación.

curl "{BASE}/api/analytics/summary?region=Tarapaca&commune=Iquique"

Los datos son informativos. No utilices la API para diagnosticar, prescribir o sustituir medicamentos automáticamente.
